# Navi - 户外卫星地图与轨迹记录

基于 Flutter 的卫星地图定位应用，专为户外徒步、勘探、测量设计。

## 功能

### 地图
- **多图源切换**：ArcGIS World Imagery（默认，WGS-84）、高德卫星（国内加载快，GCJ-02）、ESRI Clarity
- **离线缓存**：一键下载屏幕中心周边 1km² 范围、±2 级缩放的瓦片，断网可用
- **单线程串行下载**：防止服务器拒连，失败自动重试一次，支持取消
- **沉浸模式**：启动隐藏状态栏，异形屏渲染到挖孔区域
- **防误退**：首次返回键弹出"再按一次退出"提示，3 秒内再次操作才退出

### 定位与导航
- **设备位置**：蓝圈标记当前 GPS 位置，下方显示经纬度（可选开关）
- **指南针朝向**：真实设备朝向三角，以蓝圈为轴公转+自转，类似钟表指针
- **方位角显示**：蓝圈上方显示 3 位角度 + 英文方位（如 `032 N`、`135 SE`）
- **经纬度开关**：左侧 `pin_drop` 按钮控制所有坐标信息显隐，默认关闭

### 跟踪录制
- **轨迹记录**：录制 GPS 轨迹，底部居中显示时间/距离
- **后台录制**：前台服务通知保活，切到微信等后台仍持续记录
- **轨迹管理**：本地保存/加载/删除轨迹文件，支持自动保存恢复
- **轨迹回放**：加载历史轨迹到地图，分段不同颜色显示

### 路径点模式
- **添加路径点**：十字准星对准目标，点击添加
- **累计里程**：每个路径点下方显示累计距离
- **撤销**：按序撤销上一个路径点（有路径点时才显示）
- **保存/清空**：保存当前路径点到已保存列表，支持多组
- **路径点命名**：点击路径点圆圈弹出编辑面板，输入名称显示在圆圈上方
- **十字准星定位**：编辑面板内可将路径点移动到准星位置
- **退出按钮红色**：醒目提示

### 测距模式
- **画线测距**：从准星位置拉线，显示距离、方位角
- **橡皮擦**：点击线条删除
- **退出按钮红色**

### 其他
- **横竖屏切换**：左侧按钮手动切换
- **军事网格模式**：叠加米制坐标网格
- **调试控制台**：查看运行日志、位置数据、模拟位置
- **安全定时器**：定位超时 20 秒自动降级显示世界地图

## 技术栈

- **Flutter** 3.x（Dart）
- **flutter_map** 瓦片地图渲染
- **geolocator** GPS 定位（前台服务后台录制）
- **flutter_device_compass** 设备指南针
- **latlong2** 地理坐标计算
- **path_provider** 本地文件存储
- **http** 瓦片下载

## 快速开始

```bash
# 安装依赖
flutter pub get

# 本地分析/测试
flutter analyze
flutter test

# 启动（连接设备或模拟器）
flutter run

# 构建
flutter build apk --release   # Android
flutter build windows --release # Windows
```

## 项目结构

```
lib/
├── main.dart          # 入口、MapPage、UI 逻辑
├── common.dart        # 模型（TrackPoint, Waypoint）、格式化工具
├── painters.dart      # CustomPainter（朝向三角、十字准星）
├── tiles.dart         # 瓦片源配置、缓存、下载引擎
└── track_io.dart      # 轨迹文件读写
```

## 注意事项

- 高德卫星图使用 GCJ-02 坐标系，与 GPS WGS-84 有偏移，建议使用 ArcGIS
- 离线下载需网络，缓存目录按图源隔离
- Web 环境下指南针不可用，定位可能超时
